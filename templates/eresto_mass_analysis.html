Panduan Penggunaan

Fitur ini memproses file eResto penjualan gabungan seluruh outlet sekaligus, dan menghasilkan file Excel analisa per outlet dalam satu klik.

1

Format File

File harus berformat .xlsx/.xls dengan kolom Outlet di kolom D, sama seperti export eResto biasa namun dengan filter "All Outlet"

2

Deteksi Otomatis

Sistem mendeteksi outlet, rentang tanggal, dan bulan secara otomatis dari data — tidak perlu input manual

3

Output ZIP

Download satu file ZIP berisi seluruh Excel per outlet. Format nama: 29. Pauh Kambar_01-30-juni-26_tahap1.xlsx

4

Master Product

Menggunakan master product list yang sama dari menu eResto Analysis

Outlet yang tidak terdaftar di sistem tetap akan diproses, namun tanpa nomor urut di nama file.
